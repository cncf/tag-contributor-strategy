## Cloud Native Computing Foundation (CNCF)

An open-source software foundation dedicated to promoting and advancing the development of cloud-native technology. 

The Cloud Native Computing Foundation (CNCF) is an organization that promotes the use of cloud native technologies, such as containers and microservices, to build and deploy modern, scalable applications.
